## 项目简介

本项目是一款基于嵌入式图形开发的农场经营类模拟游戏。玩家将扮演农场主，通过种植作物、管理病虫害、升级设备等方式经营自己的农场。游戏结合了无人机喷洒农药的玩法，提供策略性与趣味性兼具的农场体验。

## 游戏说明

### 游戏规则

游戏核心循环：买种子 -> 种植 -> 生长 -> 防治病虫害 -> 收获 -> 卖出 -> 升级

### 基本目标

- 经营农场，持续扩大可耕种范围。
- 通过种植和收获赚取金币与经验。
- 使用无人机处理病虫害，保证作物顺利成熟。
- 提升等级后解锁更高效率的经营方式。

### 作物与生长

| 项目 | 规则 |
|---|---|
| 作物种类 | 小麦、水稻、玉米 |
| 成熟时间 | 小麦约 250 刻，水稻约 300 刻，玉米约 225 刻 |
| 生长阶段 | 种子 -> 幼苗 -> 生长 -> 开花 -> 成熟 -> 就绪 |
| 生长节奏 | 每 1 秒推进一次农田生长 |
| 风险 | 长期受虫害影响可能导致作物死亡并清空 |

### 病虫害与无人机

| 项目 | 规则 |
|---|---|
| 病虫害类型 | 蚜虫、螨虫、卷叶虫、蝗虫 |
| 农药匹配 | 每种病虫害必须使用对应农药清除 |
| 巡检方式 | 玩家可切换无人机状态进行巡检 |
| 运动方式 | 巡检时跟随摇杆移动，空闲时回到起点 |
| 检测逻辑 | 同一地块检测后不重复计数 |
| 清除收益 | 无人机有对应农药时可清除病虫害并获得经验 |

### 资源与收益

- 种植会消耗种子，并获得经验。
- 收获后产物进入库存，可出售换取金币。
- 不同作物售价不同，成熟后收益不同。
- 使用农药也会获得少量经验。
- 玩家等级越高，商店折扣越大。

## 升级机制

游戏中的升级分为 7 类，分别作用于不同对象。

| 升级类型 | 作用对象 | 效果 |
|---|---|---|
| 无人机速度升级 | 无人机 | 提高移动速度 |
| 无人机载药量升级 | 无人机 | 提高药物携带上限 |
| 无人机算法升级 | 无人机 | 优化巡检路径规划 |
| 农场规模升级 | 整个农场 | 扩大可用田地范围 |
| 田地产量升级 | 单块田地 | 提高收获产量 |
| 田地成熟速度升级 | 单块田地 | 缩短成熟时间 |
| 田地耐虫性升级 | 单块田地 | 降低受虫害影响压力 |

### 升级规则

- 无人机升级和农场升级影响整体经营能力。
- 田地升级只对指定地块生效。
- 田地升级为下一季生效，不会立刻改变当前作物。
- 购买升级需要消耗金币，并受等级折扣影响。

## 玩家成长

- 玩家初始等级较低，需要通过种植、收获、喷药积累经验。
- 经验达到阈值后自动升级。
- 等级提升会带来商店折扣，让后续经营更容易。
