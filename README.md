# 农田无人机喷洒农药模拟系统（无人机农场游戏）

## 项目简介

本项目是一款基于嵌入式图形开发的农场经营类模拟游戏。玩家将扮演农场主，通过种植作物、管理病虫害、升级设备等方式经营自己的农场。游戏结合了无人机喷洒农药的创新玩法，提供策略性与趣味性兼具的农场体验。

## 主要功能

- **农场经营与作物管理**  
  种植多种作物，管理作物从播种到成熟的完整生长周期，收获作物获得金币与经验。

- **病虫害防治与无人机系统**  
  作物生长过程中会随机出现病虫害，玩家可操控无人机勘察农田、配置农药并自动规划喷洒路径，消除病害并获取经验奖励。

- **商店交易与成长系统**  
  使用金币购买种子、农药及农场升级道具。随着等级提升，商店购物折扣增加，经营更加轻松。

- **实时数据展示**  
  界面实时显示金币、经验、等级、无人机状态、作物生长进度等关键信息，方便玩家掌握农场动态。

## 开发平台与设备

- 基于 GD32H7 嵌入式开发板
- 图形显示屏、摇杆、按键、扬声器
- SD 卡（用于数据持久化存储）
